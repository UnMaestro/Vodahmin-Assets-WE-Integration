# Vodahmin Assets WE integration

Hello and Hallo. This is WriteEverywhere Module that adds dynamic Text to Vodahmins VAG station assets.

### Functions
- Works **automaticly**, you don't need any WE knowledge whatsoever!
- Dynamic working Bus/Tram Stop Names

---

##  Requirements
- [Write Everywhere](https://mods.paradoxplaza.com/mods/92908/Windows) – required  
- [VAG Bus Stop Set](https://mods.paradoxplaza.com/mods/132358/Any)
- [VAG Tram Stop Set](https://mods.paradoxplaza.com/mods/132354/Any)
